# 🚶‍♂️ Route Replanning for Mobility-Impaired Navigation

> **교통약자를 위한 실시간 경로 재탐색 길찾기 프로그램**

동적으로 변화하는 도로 환경에서 **교통약자(휠체어 이용자, 노약자, 유모차 이용자 등)**가 안전하게 이동할 수 있도록 실시간으로 최적의 경로를 재탐색하는 시뮬레이션입니다.

---

## 📖 프로젝트 소개

일반적인 길찾기 서비스는 출발 시 계산한 경로를 제공하지만, 실제 도로에서는 다양한 장애물이 발생합니다.

본 프로젝트는 다음과 같은 상황을 고려합니다.

- 🚧 갑작스럽게 생기는 공사 구간
- 🚗 이동하는 차량
- 🚦 신호가 바뀌는 횡단보도
- 🛗 경사로

사용자가 이동하는 동안 이러한 환경 변화가 발생하면 **A\* (A-Star) 알고리즘을 이용하여 실시간으로 새로운 최적 경로를 계산**합니다.

---

## 🎯 프로젝트 목표

- 교통약자를 위한 안전한 이동 지원
- 실시간 장애물 인식
- 경로 재탐색 알고리즘 구현
- 다양한 장애 요소를 고려한 길찾기 시뮬레이션

---

## ✨ 주요 기능

### ✅ 실시간 경로 탐색 (A*)

- 시작 지점에서 목적지까지 최단 경로 계산
- Manhattan Distance 기반 휴리스틱 사용

---

### 🚧 동적 장애물

시간이 지남에 따라 새로운 장애물이 생성됩니다.

- 공사 구역
- 이동 불가능한 장애물

장애물이 기존 경로를 막을 경우 즉시 새로운 경로를 계산합니다.

---

### 🛗 경사로

경사로는 이동이 가능하지만

- 이동 속도가 감소
- 일반 도로보다 높은 이동 비용

을 갖도록 구현되어 있습니다.

---

### 🚗 이동 차량

차량이 일정한 경로를 반복 이동합니다.

차량이 지나가는 위치는 일시적으로 이동 불가능한 구역으로 판단하여
경로 탐색에 반영됩니다.

---

### 🚦 신호등 횡단보도

횡단보도는

- 🔴 빨간불
- 🟢 초록불

상태를 반복하며,

빨간불에는 통행이 불가능한 장애물로 처리됩니다.

---

### 🎯 목적지 변경

사용자가 바닥을 클릭하면

새로운 목적지로 즉시 경로를 다시 계산합니다.

---

## 🧠 사용 알고리즘

### A* (A-Star)

사용한 휴리스틱

```
h(n) = |x₁-x₂| + |y₁-y₂|
```

특징

- 최단 경로 탐색
- 장애물 회피
- 빠른 탐색 속도
- 실시간 재계산 가능

## 💡 기대 효과

본 프로젝트는 일반적인 최단 경로 탐색을 넘어,

교통약자의 이동 환경을 고려한 **접근성 중심 길찾기 시스템**을 제안합니다.

실시간으로 변화하는 도로 환경을 반영하여 보다 안전하고 효율적인 이동을 지원하며, 향후 실제 지도 API와 연계할 경우 스마트 내비게이션 서비스로 확장할 수 있습니다.
